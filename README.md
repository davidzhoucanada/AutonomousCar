# AutonomousCar
Autonomous car made from store-bought RC car. Controlled manually with an Android app via Bluetooth and an Arduino

## Built With
* Store-bought RC car
* Arduino Uno
* Motor driver
* Bluetooth module
* Ultrasonic sensors
* *A lot* of electrical tape
